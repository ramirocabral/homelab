## Environment Variables

| Variable      | Description                       |
|---------------|-----------------------------------|
| `WAREZ_ROOT`  | Root directory for Warez stuff.   |
| `DATA_ROOT`   | Root directory for personal data. |
| `SMB_USERNAME`| Username for SMB shares.          |
| `SMB_PASSWORD`| Password for SMB shares.          |
